# concurrent2

## connecting to stoker
connect using openVPN

`ssh -X username@stoker.scss.tcd.ie`

## copying files to stoker
while not ssh'ed

`scp <directory(ies)> username@stoker.scss.tcd.ie:~/<directory>`
